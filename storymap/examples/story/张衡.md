# 张衡

## 人物档案

### 基本信息
- **姓名**：张衡
- **时代**：东汉
- **出生**：公元78年，南阳郡西鄂县（今河南省南阳市）（一说公元78年，存疑）
- **去世**：公元139年，洛阳（今河南省洛阳市）
- **享年**：62岁
- **主要身份**：天文学家、数学家、发明家、地理学家、文学家
- **历史地位**：中国东汉时期伟大的全才，在天文学、地震学、机械技术、文学等领域成就卓著。
- **主要成就**：
    1. 发明**浑天仪**，系统阐述**浑天说**宇宙理论。
    2. 发明**候风地动仪**，是世界最早的验震器。
    3. 著有天文学著作《**灵宪**》，解释月食成因。
    4. 撰写《**二京赋**》《**归田赋**》等名篇，为汉赋代表作家。
    5. 绘制地形图，精通算学。

### 生平概述
张衡，字平子，生于东汉南阳郡。他少年游学，通贯五经六艺，尤精天文历算。曾任太史令，主持观测天象、编订历法，期间发明**浑天仪**与**候风地动仪**，将中国古代天文学与地震监测技术推向高峰。晚年出任河间相，治政严明。他一生横跨科学与人文，既是严谨的科学家，又是才华横溢的文学家，其成就体现了东汉时期科技与文化的繁荣。

## 人生历程与重要地点（按时间顺序）

### 🟢 出生地：南阳郡西鄂县
- **时间**：公元78年
- **停留时间**：约15年
- **地点**：南阳郡西鄂县（今河南省南阳市北）
- **事件**：出生于当地名门，少年时代在此度过，接受启蒙教育。
- **意义**：奠定了其早期学识基础，故乡的文化氛围对其成长有深远影响。

### 📍 重要地点：三辅地区（游学）
- **时间**：约公元93年 - 公元95年
- **停留时间**：约2-3年
- **地点**：三辅地区（今陕西省中部，以长安为中心）
- **事件**：离乡游学，进入当时的学术文化中心，考察民情风俗，积累社会经验。
- **意义**：开阔了眼界，深入了解社会现实，为其创作《二京赋》积累了丰富素材。
- **名篇名句**：《二京赋》：“水所以载舟，亦所以覆舟。”

### 📍 重要地点：洛阳（太学求学）
- **时间**：约公元95年 - 公元100年
- **停留时间**：约5年
- **地点**：洛阳（今河南省洛阳市）
- **事件**：进入国家最高学府**太学**学习，“通《五经》，贯六艺”。
- **意义**：系统学习了儒家经典与各种技艺，为其成为通才奠定了坚实的学术基础。

### 📍 重要地点：南阳郡（任主簿）
- **时间**：约公元100年 - 公元111年
- **停留时间**：约11年
- **地点**：南阳郡（今河南省南阳市）
- **事件**：应太守鲍德之邀，出任南阳郡主簿，掌管文书。期间专心钻研学问。
- **意义**：在处理政务之余，进行学术研究和文学创作，完成了《二京赋》的初稿。

### 📍 重要地点：洛阳（首任太史令）
- **时间**：公元111年 - 公元121年
- **停留时间**：约10年
- **地点**：洛阳（今河南省洛阳市）
- **事件**：被朝廷征召，拜为**郎中**，后升任**太史令**。主持天文观测与历法修订。
- **意义**：进入国家科技核心机构，开始系统进行天文学研究，为发明**浑天仪**和撰写《**灵宪**》创造了条件。
- **名篇名句**：《灵宪》：“月光生于日之所照。”

### 📍 重要地点：洛阳（复任太史令，发明地动仪）
- **时间**：公元126年 - 公元133年
- **停留时间**：约7年
- **地点**：洛阳（今河南省洛阳市）
- **事件**：在短暂调任他职后，复任**太史令**。于公元132年，发明并奏上**候风地动仪**。
- **意义**：这是其科学生涯的巅峰成就之一，标志着世界地震学研究的开端。
- **名篇名句**：《后汉书·张衡传》记载地动仪：“如有地动，尊则振龙，机发吐丸，而蟾蜍衔之。”

### 📍 重要地点：河间国（任河间相）
- **时间**：公元136年 - 公元138年
- **停留时间**：约2年
- **地点**：河间国（今河北省献县东南）
- **事件**：出任河间相。当时河间王刘政骄奢，地方豪强不法。张衡到任后，“治威严，整法度”，打击豪强，清理冤狱。
- **意义**：展现了其卓越的政治才能和整顿吏治的决心，政绩卓著。

### 🔴 去世地：洛阳
- **时间**：公元139年
- **停留时间**：晚年
- **地点**：洛阳（今河南省洛阳市）
- **事件**：从河间相任上被征召入朝，拜为**尚书**。同年病逝于洛阳。
- **意义**：一代科学巨匠与文学大家陨落，归葬于故乡南阳西鄂。
- **名篇名句**：《归田赋》：“超埃尘以遐逝，与世事乎长辞。”

## 人教版教材知识点
### 语文（课文/词作）
- 课文/词作：**《归田赋》**（节选）、**《二京赋》**（简介）
- 核心要点：
    - **重点**：**《归田赋》** 是中国文学史上第一篇以**田园隐居**为主题的抒情小赋，开后世田园文学之先声。
    - 艺术手法上，《二京赋》继承了汉大赋**铺采摘文**的特点，但讽谏意味更强，反映了作者对社会现实的关切。
    - **重点**：张衡的赋作体现了**由大赋向抒情小赋过渡**的文学史趋势，其作品兼具**宏大叙事**与**个人抒情**。
    - 《归田赋》中“**仰飞纤纤，俯钓长流**”等句，描绘了恬淡的田园乐趣，表达了对官场纷扰的厌倦。
    - **重点**：张衡的文学成就与其科学精神相通，都体现了**求真**与**务实**的倾向，其赋作内容扎实，不同于纯粹的辞藻堆砌。

### 历史（史实/人物定位）
- 关键史实：
    - **重点**：张衡于公元132年发明**候风地动仪**，是世界科技史上的重大事件，比西方同类仪器早约1700年。
    - 他担任**太史令**长达十余年，是东汉官方天文学的主要负责人，主持观测、记录天象。
    - **重点**：张衡是**浑天说**的代表人物，其著作《**灵宪**》系统论述了宇宙起源、天地结构及日月星辰的运行规律。
    - 他绘制过**地形图**，在数学上对**圆周率**也有研究。
- 核心要点：
    - **重点**：张衡是中国历史上罕见的**全才型科学家**，其成就横跨天文、地震、机械、地理、数学、文学等多个领域。
    - 他的科学成就（如地动仪）体现了东汉时期**经验科学**与**精密仪器制造**的高超水平。
    - **重点**：张衡的生平与成就，是**东汉时期综合国力与文化科技繁荣**的一个缩影。
    - 他“**不慕当世**”的淡泊性格与执着于**探索自然规律**的精神，是其留给后世的重要精神遗产。
    - **重点**：后世常将张衡与**祖冲之**、**郭守敬**等并列，视为中国古代科学家的杰出代表。
