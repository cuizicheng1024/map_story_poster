# 赵树理

## 人物档案

### 基本信息
- **姓名**：赵树理
- **时代**：中国现代（清末至新中国）
- **出生**：公元1906年9月24日，山西省沁水县尉迟村（今山西省晋城市沁水县）
- **去世**：公元1970年9月23日，北京市
- **享年**：64岁
- **主要身份**：作家、小说家、人民艺术家
- **历史地位**：中国现代文学“山药蛋派”的开创者与代表人物，其作品深刻反映了中国农村的社会变革与农民生活。
- **主要成就**：
    1. 创立了具有鲜明民族化、大众化风格的“山药蛋派”文学流派。
    2. 代表作《小二黑结婚》《李有才板话》等成为解放区文学的典范。
    3. 长期致力于用农民喜闻乐见的语言和形式进行文学创作。
    4. 对中国现代农村题材文学的发展产生了深远影响。

### 生平概述
赵树理出生于贫苦农民家庭，深受民间文艺熏陶。他经历了新文化运动、抗日战争、解放战争和新中国成立等重大历史时期。其文学生涯始于20世纪30年代，成熟于40年代的延安文艺座谈会后。他始终坚持深入农村生活，以通俗易懂、幽默风趣的语言，创作了大量反映农村阶级斗争、社会改革和农民思想转变的作品，被誉为描写中国农村的“铁笔”和“圣手”。他的作品真实记录了20世纪中国农村的巨大变迁。

## 人生历程与重要地点（按时间顺序）

### 🟢 出生地：尉迟村
- **时间**：1906年
- **停留时间**：童年及少年时期（约14年）
- **地点**：山西省沁水县尉迟村（今山西省晋城市沁水县嘉峰镇尉迟村）
- **事件**：出生于贫农家庭，原名赵礼饱。自幼随祖父识字，接触上党梆子、民歌、鼓词等民间艺术。
- **意义**：奠定了其文学创作的民间文化基础和对农民生活的深刻理解。

### 📍 重要地点：长治
- **时间**：1925年 - 1928年
- **停留时间**：约3年
- **地点**：山西省潞安府（今山西省长治市）
- **事件**：考入山西省立第四师范学校（长治师范），开始接触新文学和五四新思想。
- **意义**：接受了现代教育，思想发生重要转变，萌生了用文学改造社会的志向。

### 📍 重要地点：太原
- **时间**：1929年 - 1936年（期间多次往返）
- **停留时间**：断续居住数年
- **地点**：山西省太原府（今山西省太原市）
- **事件**：因参与学生运动被开除、被捕。出狱后在此当过小学教员、职员，并开始尝试写作，创办《山西党讯》副刊。
- **意义**：在动荡中坚持创作和编辑工作，早期作品在此发表，文学与革命思想进一步结合。

### 📍 重要地点：阳城、沁水
- **时间**：1937年 - 1939年
- **停留时间**：约2年
- **地点**：山西省阳城县、沁水县（今山西省晋城市阳城县、沁水县）
- **事件**：抗战爆发后加入牺盟会，从事抗日宣传和基层工作，担任过区长。
- **意义**：直接参与农村抗日动员，积累了丰富的农村工作和生活素材。

### 📍 重要地点：长治（第二次重要停留）
- **时间**：1940年 - 1942年
- **停留时间**：约2年
- **地点**：山西省长治县（今山西省长治市）
- **事件**：在《黄河日报》（路东版）、《人民报》从事编辑工作，创作大量通俗文艺作品。
- **意义**：其大众化、通俗化的文艺实践在根据地产生广泛影响。

### 📍 重要地点：延安
- **时间**：1943年5月 - 1945年
- **停留时间**：约2年半
- **地点**：陕西省延安府（今陕西省延安市）
- **事件**：抵达延安。在毛泽东同志《在延安文艺座谈会上的讲话》精神指引下，创作出成名作《小二黑结婚》和《李有才板话》。
- **意义**：创作进入黄金期，作品获得巨大成功，确立了其文学地位和“山药蛋派”的雏形。
- **名篇名句**：《小二黑结婚》：“小二黑好比一枝花，可惜开在粪堆旁。”；《李有才板话》：“村长阎恒元，一手遮住天。”

### 📍 重要地点：武安、邢台
- **时间**：1946年 - 1948年
- **停留时间**：约2年
- **地点**：河北省武安县、邢台县（今河北省邯郸市武安市、邢台市）
- **事件**：在晋冀鲁豫边区文联工作，主编《新大众》报。创作长篇小说《李家庄的变迁》。
- **意义**：继续深入农村，作品反映了解放战争时期农村的深刻变革。
- **名篇名句**：《李家庄的变迁》：“这地方的老百姓，怕官怕得厉害。”

### 📍 重要地点：北京
- **时间**：1949年 - 1965年（主要居住时期）
- **停留时间**：约16年
- **地点**：北平/北京（今北京市）
- **事件**：新中国成立后，历任《说说唱唱》主编、《人民文学》编委、中国曲艺家协会主席、中国作家协会理事等职。创作《登记》《锻炼锻炼》《三里湾》等作品。
- **意义**：成为全国性文艺界领导人，但坚持深入北京郊区及山西农村体验生活，创作反映农业合作化等新时代问题的作品。
- **名篇名句**：《登记》：“罗汉钱，金光闪，越看越爱越心酸。”；《锻炼锻炼》：“‘小腿疼’和‘吃不饱’这两个外号，比她们的学名还响亮。”

### 📍 重要地点：太原（第二次重要停留）
- **时间**：1965年 - 1966年
- **停留时间**：约1年
- **地点**：山西省太原市（今山西省太原市）
- **事件**：调回山西，兼任晋城县委副书记，深入基层。
- **意义**：试图回归熟悉的农村生活环境，但不久后政治风暴来临。

### 🔴 去世地：北京
- **时间**：1970年9月23日
- **停留时间**：生命最后几年（1966-1970）
- **地点**：北京（今北京市）
- **事件**：在“文化大革命”中遭受残酷迫害，含冤去世。
- **意义**：一位始终为农民写作的人民作家悲剧性地陨落。1978年获得平反。

## 人教版教材知识点
### 语文（课文/词作）
- 课文/词作：**《小二黑结婚》**（节选）、**《李有才板话》**（节选）、《老杨同志》（节选自《李有才板话》）
- 核心要点：
    - **重点**：作品深刻体现了 **延安文艺座谈会讲话** 后解放区文学 **为工农兵服务** 的创作方向。
    - **重点**：成功塑造了 **小二黑、小芹** 等新一代农民形象，以及 **二诸葛、三仙姑** 等落后农民的典型。
    - **重点**：语言艺术极具特色，采用经过提炼的 **农民口语**，幽默风趣，通俗易懂，形成了 **“山药蛋派”** 的鲜明风格。
    - 善于通过 **日常生活矛盾**（如婚姻自由、政权建设）反映农村社会变革的重大主题。
    - 叙事结构借鉴了 **中国传统说唱文学** 和 **章回小说** 的手法，故事性强。
    - 作品具有浓郁的 **山西乡土气息** 和地方色彩。

### 历史（史实/人物定位）
- 关键史实：
    - **重点**：赵树理是 **延安文艺座谈会讲话** 精神的杰出实践者，其创作方向代表了解放区文艺的新气象。
    - 其作品是研究 **抗日根据地** 和 **解放区农村社会变革**（如减租减息、婚姻法、政权建设）的重要文学史料。
    - **重点**：作为 **“山药蛋派”** 的创始人，他影响了马烽、西戎等一批山西作家，形成了当代文学史上重要的流派。
- 核心要点：
    - **重点**：赵树理的文学道路体现了 **知识分子与工农群众相结合** 的时代要求，他始终以 **“地摊文学家”** 自居。
    - **重点**：他的创作核心是 **“问题小说”**，旨在通过文学作品揭示并推动解决农村实际工作中的问题。
    - 其文学观强调 **民族形式** 与 **大众化**，反对脱离群众的“欧化”语言和形式。
    - **重点**：他的人生与创作命运，折射出 **20世纪中国农村变迁** 与 **文艺政策演变** 的复杂互动关系。
