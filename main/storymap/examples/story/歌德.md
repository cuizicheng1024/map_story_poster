# 歌德

## 人物档案

### 基本信息
- **姓名**：约翰·沃尔夫冈·冯·歌德
- **时代**：18-19世纪（德国启蒙运动、狂飙突进运动、古典主义与浪漫主义时期）
- **出生**：公元1749年8月28日，美因河畔法兰克福（今德国法兰克福）
- **去世**：公元1832年3月22日，魏玛（今德国图林根州魏玛市）
- **享年**：83岁
- **主要身份**：诗人、剧作家、小说家、自然科学家、政治家
- **历史地位**：德国最伟大的作家之一，世界文学巨匠，其作品与思想深刻影响了欧洲文学与思想进程。
- **主要成就**：创作诗剧《浮士德》；小说《少年维特之烦恼》；提出形态学与色彩理论；长期主持魏玛公国政务，推动文化艺术发展。

### 生平概述
歌德生于法兰克福一个富裕的市民家庭，早年学习法律，但志在文学与艺术。他参与发起“狂飙突进”运动，以《少年维特之烦恼》蜚声文坛。后受邀至魏玛公国，身居枢密顾问要职，涉足政治、科学、军事管理等多个领域。中年曾旅居意大利，古典艺术对其影响深远。晚年深居简出，潜心创作，最终完成毕生巨著《浮士德》。他的一生横跨多个时代，作品融合了启蒙理性与浪漫情感，成为连接德国与世界文化的重要桥梁。

## 人生历程与重要地点（按时间顺序）

### 🟢 出生地：美因河畔法兰克福
- **时间**：1749年 - 1765年
- **停留时间**：约16年
- **地点**：美因河畔法兰克福（今德国法兰克福）
- **事件**：出生于富裕的市民家庭，接受良好早期教育，阅读广泛，对民间戏剧产生兴趣。
- **意义**：塑造了其世界观的基础，法兰克福自由市的环境培养了他对政治与社会的早期认知。

### 📍 重要地点：莱比锡
- **时间**：1765年 - 1768年
- **停留时间**：约3年
- **地点**：莱比锡（今德国萨克森州莱比锡市）
- **事件**：遵从父命学习法律，但主要兴趣在文学与艺术创作，结识了早期的重要朋友与导师。
- **意义**：创作风格转向洛可可式，出版首部诗集；因病休学，人生首次重大挫折。
- **名篇名句**：《情人的脾气》：早期洛可可风格诗歌。

### 📍 重要地点：斯特拉斯堡
- **时间**：1770年 - 1771年
- **停留时间**：约1年
- **地点**：斯特拉斯堡（今法国斯特拉斯堡）
- **事件**：继续学业并获得法学博士学位；结识思想家赫尔德，受其影响收集民歌，文学观念发生巨变。
- **意义**：“狂飙突进”运动思想形成的关键期，开始创作具有鲜明反叛精神的作品。
- **名篇名句**：《五月之歌》：自然与情感交融的抒情诗。

### 📍 重要地点：韦茨拉尔
- **时间**：1772年
- **停留时间**：数月
- **地点**：韦茨拉尔（今德国黑森州韦茨拉尔市）
- **事件**：在帝国最高法院实习，结识友人并经历无望的爱情，此段经历成为《少年维特之烦恼》的素材。
- **意义**：个人情感体验与时代苦闷结合，催生了其第一部具有世界影响力的小说。
- **名篇名句**：《少年维特之烦恼》：“哪个少年不钟情，哪个少女不怀春。”

### 📍 重要地点：魏玛（首次抵达与任职）
- **时间**：1775年 - 1786年
- **停留时间**：约11年
- **地点**：魏玛（今德国图林根州魏玛市）
- **事件**：受魏玛公爵卡尔·奥古斯特邀请，担任枢密顾问，主管军事、交通、财政等实务。
- **意义**：从狂飙突进的青年作家转变为务实的政治家与管理者，进入人生新阶段。

### 📍 重要地点：意大利
- **时间**：1786年 - 1788年
- **停留时间**：约2年
- **地点**：威尼斯、罗马、西西里岛等地（今意大利）
- **事件**：秘密前往意大利，研究古典艺术与自然科学，完成多部剧本，艺术观转向古典主义。
- **意义**：“意大利之旅”是其精神与艺术的重生，确立了“高贵的单纯，静穆的伟大”的美学理想。
- **名篇名句**：《意大利游记》（日记与书信集）。

### 📍 重要地点：魏玛（与席勒合作时期）
- **时间**：1794年 - 1805年
- **停留时间**：约11年
- **地点**：魏玛（今德国图林根州魏玛市）
- **事件**：与席勒建立深厚友谊并密切合作，共同主办《季节女神》杂志，推动德国古典文学高峰。
- **意义**：德国文学的“古典主义”时期，两人合力创作，成就文学史佳话。
- **名篇名句**：《浮士德》（片段）；《威廉·迈斯特的学习时代》。

### 📍 重要地点：魏玛（晚年与《浮士德》完成）
- **时间**：1806年 - 1832年
- **停留时间**：约26年
- **地点**：魏玛（今德国图林根州魏玛市）
- **事件**：经历拿破仑战争，晚年沉浸于科学研究与《浮士德》第二部的创作，直至生命最后。
- **意义**：完成思想与艺术的集大成之作，使魏玛成为欧洲文化圣地。
- **名篇名句**：《浮士德》：“凡是自强不息者，到头我辈终能救。”

### 🔴 去世地：魏玛
- **时间**：1832年3月22日
- **停留时间**：（同上，晚年居所）
- **地点**：魏玛（今德国图林根州魏玛市）
- **事件**：在完成《浮士德》手稿封存后不久，因病逝世。
- **意义**：标志着德国古典文学时代的终结，其故居与陵墓成为重要文化纪念地。

## 人教版教材知识点
### 语文（课文/词作）
- 课文/词作：**《浮士德》**（选段）、**《少年维特之烦恼》**（节选）、《五月之歌》（诗歌）。
- 核心要点：
    - **重点**：**《浮士德》** 的核心主题是 **自强不息** 的探索精神，通过主人公追求真理、爱情、事业、美与社会理想的历程展现。
    - **重点**：**《少年维特之烦恼》** 是 **狂飙突进运动** 的代表作，体现了对 **个性解放** 和 **自然情感** 的强烈呼唤。
    - 作品中大量运用 **象征** 手法，如浮士德与魔鬼的契约象征人类对欲望与知识的矛盾。
    - 诗歌风格从早期的 **洛可可** 风格，发展到狂飙突进的 **激情澎湃**，再到古典时期的 **和谐静穆**。
    - **重点**：歌德善于将 **个人经历** 与 **时代精神** 相融合，使作品具有深刻的普遍性。
    - 语言艺术上，其作品兼具 **哲理性** 与 **抒情性**，对德语文学语言的规范化有巨大贡献。

### 历史（史实/人物定位）
- 关键史实：
    - **重点**：歌德是 **狂飙突进运动** 的核心人物，该运动是德国启蒙运动的高潮，强调情感、天才与反抗。
    - 1775年后，他受邀进入 **魏玛公国** 政府，成为开明 **贵族政治** 的实践者。
    - **重点**：他与 **席勒** 的合作（1794-1805）共同开创了德国文学的 **“古典主义”** 时期。
    - 他的 **自然科学研究**（如色彩学、形态学）是其理性与经验主义思想的重要体现。
- 核心要点：
    - **重点**：歌德的一生是 **启蒙理性** 与 **浪漫情感** 之间的桥梁，其思想具有 **过渡时代** 的复杂性。
    - 他不仅是文学家，更是 **文艺复兴式** 的全才，在政治、科学、艺术管理等领域均有建树。
    - **重点**：**魏玛** 因歌德与席勒等人的存在，成为18世纪末至19世纪初 **德意志文化的中心**。
    - 对他的评价历来多元：既是 **体制内的改革者**，也是 **精神上的永恒漫游者**。
