# 🗺️ StoryMap：中学老师的历史人物足迹互动课件生成器

还在为备课时梳理复杂的历史人物行踪而头秃吗？
还在为学生分不清“辛弃疾到底去过哪儿”而苦恼吗？
**StoryMap** 专为中学历史、语文老师打造，只需输入一个名字，就能自动生成一份精美的**交互式地图课件**。

### 🎯 为什么推荐给老师们？
- **💡 辅助高效备课**：自动抓取人物生平，省去翻阅史料、查找古地名的繁琐过程。
- **📍 直观展现轨迹**：将文字叙述转化为地图足迹，人物的一生流转一目了然。
- **📚 跨学科教学**：完美契合“大语文”、“大历史”教学理念，在地图中讲诗词，在地理中读历史。
- **✨ 吸引学生注意力**：生成可交互的网页，支持时间轴点击和事件弹窗，让课堂不再枯燥。

---

### ⚙️ 后台自动流程（老师们无需关心技术细节）
项目在后台通过一套智能化的流程，为您打理好一切：

1. **输入姓名**：您只需输入想讲的人物（如：`辛弃疾`）。
2. **生成故事**：AI 会自动整理人物档案、人生关键足迹（时间、地点、事件）以及教材中的考点。
3. **定位古地名**：系统会自动把“润州”、“京口”等古地名精准对应到现代地图坐标。
4. **生成课件网页**：最终打包成一个漂亮的 HTML 网页，包含时间轴和地图动效，直接用浏览器就能打开。

---

### 🚀 如何开始使用

#### 1. 🛠️ 准备环境
本工具推荐使用 **Python 3.11**。请在终端执行以下命令安装依赖：

```bash
pip install -r maptoposter/requirements.txt
```

#### 2. 🔑 配置 AI 钥匙 (可选)
如果需要 AI 自动生成内容，请在项目根目录创建一个 `.env` 文件并填入：
```bash
API_KEY=您的_API_KEY
BASE_URL=https://api.openai.com/v1
MODEL=gpt-4o-mini
```
*温馨提示：如果没有配置，系统会默认以“辛弃疾”为例为您演示效果。*

#### 3. 💻 一键生成课件
在终端输入这行简单的命令：

```bash
python3 auto_generate.py --name "辛弃疾"
```

#### 4. 📄 查看并分发课件
生成的网页会保存在：
`map_story/storymap/examples/story_map/`

脚本运行完后，会自动在浏览器里为您打开这个课件。您可以直接把这个 HTML 文件拷给学生或在课堂上投屏展示。

### ✨ 我们的亮点
- **📦 离线也能演示**：生成的课件是一个独立的网页文件，不需要安装任何软件，没网也能讲。
- **🏷️ 古地名不迷路**：内置专业的古今地名索引，精准解决“这个地方现在在哪”的问题。
- **🎓 紧贴教材考点**：AI 生成的内容会自动包含“人教版教材知识点”，备课更省心。
- **📊 更多可能**：如果您想进一步研究，还可以导出 GeoJSON 数据进行地理分析。
